树莓派入门

操作系统安装

下载操作系统镜像

  • 树莓派操作系统下载地址：树莓派操作系统下载地址

其中，Raspbian是官方的操作系统，点击进去之后就是如下界面，推荐安装带桌面的操作系统。

格式化SD卡

使用软件SD Memory Card formatter格式化SD卡，软件下载地址：SD Memory Card formatter下载地址

完后软件安装后，将内存卡插在电脑上，打开该软件，选择要格式化的内存卡，不需其它设置，点击Format即可。

安装操作系统

文章中用到的烧录软件 balenaEtcher，软件下载地址：balenaEtcher下载地址

如果你电脑是windows系统的话，就选上面两个，不同的是第一个是安装包，第二个是免安装包。

打开软件的界面很简洁，从左到右依次进行选择即可。

点击Flash！进行系统烧录，完成操作系统安装后，将SD插到树莓派开发板上，通电开机即可！

树莓派远程连接

SSH远程连接

  • 打开树莓派SSH服务

    通过命令：sudo raspi-config 进入树莓派系统自带的配置工具，树莓派很多必要功能都需要它来设置。

  • 远程登录工具putty下载，下载地址：putty下载地址，选择对应的系统版本下载即可

  • 完成软件安装后打开，输入树莓派ip地址，端口号设置为22

  • 登录默认用户名为pi，密码为raspberry

此时，树莓派已经登录成功，不过只能进行无界面化的操作，要进行界面化操作，可以在这个基础上展开。

  • 安装xrdp

    通过命令：sudo apt-get install xrdp 在树莓派上安装xrdp

    在安装过程中遇到了无法定位软件包的错误：

    原因：本地没有该功能的资源；

    解决方法：更新资源

    更新资源后，再次安装xrdp

  • 远程桌面连接

    在这里，我们使用windows自带的远程桌面连接功能mstsc连接树莓派，win+r->mstsc打开，输入树莓派IP地址后连接树莓派，然后输入树莓派用户名和密码

常见问题及解决方法

vim方向键变ABCD输入

  • 更新资源

  • 删除vim

  • 重新安装vim

资源更新速度慢

  • 备份文件

  • 修改软件更新源，执行如下命令：sudo nano /etc/apt/sources.list

    将第一行中的官方软件源更新地址修改为中科大的软件更新源地址

  • 修改系统更新源，执行如下命令：sudo nano /etc/apt/sources.list.d/raspi.list

    将第一行中的官方系统源更新地址修改为中科大的系统更新源地址

  • 同步更新源&更新升级安装软件包
