结构体对齐原则

#pragma pack(n)对齐数据结构大小的计算方法：

[1] 程序员可以通过预编译命令#pragma pack(n)，n=1,2,4,8,16改变对齐系数；

[2] 结构体中每个数据都应满足：首地址%min{该数据自身长度，n} =0；

[3] 整个结构体长度对齐，即：(本条规则为个人总结，程序验证) 结构体总长度%min{ max{结构体中占用内存最大的数据类型的长度}，n } =0；
