和数字相关的函数

  • 1 数据类型

    bool()：布尔型

    int()：整型

    float()：浮点型

    complex()：浮点型

  • 2 进制转换

    bin()：将所给参数转换为二进制

    oct()：将所给参数转换为八进制

    hex()：将所给参数转换为十六进制

  • 3 数学运算

    abs()：计算绝对值

    divmode(a, b)：计算a除以b的商和余数

    round()：四舍五入

    pow(a, b)：计算a的b次幂

    sum()：求和

    min()：求最小值

    max()：求最大值
